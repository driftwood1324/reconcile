# Vendored fonts

This directory vendors the Latin variable WOFF2 subsets used by the app so `next build` does not fetch Google Fonts at build time.

- `inter-latin-variable.woff2` — Inter variable font, Latin subset, sourced from Google Fonts.
- `lora-latin-variable.woff2` — Lora variable font, Latin subset, sourced from Google Fonts.

Both font families are distributed under the SIL Open Font License 1.1 via Google Fonts.
